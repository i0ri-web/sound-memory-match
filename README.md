# Sound Memory Match (音で合わせる神経衰弱)

## ゲームの説明
4x4（全16マス）のタイルをクリックすると、それぞれ異なる電子音が鳴ります。
同じ音が鳴るタイルを2枚連続で見つけて（ペアにして）いく、音感と記憶力が試される神経衰弱ゲームです。
すべてのペア（8組）を見つけるまでの手数（Moves）を競います。

## 遊ぶには
以下のリンクから、ブラウザですぐに遊ぶことができます。
👉 **[遊んでみる (GitHub Pages)](https://i0ri-web.github.io/sound-memory-match/)**

※ 初回クリック時にブラウザから音声再生の許可が求められる場合があり、それ以降音が鳴るようになります。

## このゲームの作成に使用したプロンプト
このゲームは、以下のプロンプトをAIに指示することで生成されました。

> 4x4のクリックすると音が鳴るマスの、同じ音を揃えて消す神経衰弱を作って

### 技術的な特徴
- Web Audio API を用いてブラウザ上でリアルタイムに音声を生成しているため、外部の音声ファイル（mp3やwavなど）は不要です。
- Vanilla JS, HTML, CSS (1ファイルの `index.html`) のみで構築されています。
